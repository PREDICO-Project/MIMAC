Filters Parameters

 1%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
 2%%           FILTERS PARAMETERS                 %%
 3%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
 4
 5:P Filters True
 6# Apply Filter1, Kill all electrons
 7:P ApplyF1 True
 8# Apply Filter2, Kill secondary tracks
 9:P ApplyF2 False
10# Apply Filter3, Kill scattered photons
11:P ApplyF3 False
12# Apply Filter4, Kill photons at the jaws
13:P ApplyF4 False
14# Apply Filter5, Kill photons at detectorEND
15:P ApplyF5 False

  • ApplyF1: If Yes Kill all electrons when they are generated (highly recommended).

  • ApplyF2: If Yes Kill all secondary tracks.

  • ApplyF3: If Yes Kill all scatered photons.

  • ApplyF4: If Yes Kill all non-scattered photons (not recommended).

  • ApplyF5: If Yes Kill all photons that goes through the jaws (recommended if Jaws are defined).

  • ApplyF6: If Yes Kill all photons at the end of the detector.

Note

The Filter 2 and 3 are recommended if you want a faster simulation. However, using this Filters and the anti scatter grid could be redundant.
