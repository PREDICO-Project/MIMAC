Anti-scatter Grid Parameters

1%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
2%%       ANTI-SCATTER GRID PARAMETERS           %%
3%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
4
5:P UseAntiScatterGrid True
6:P GridRatio 5.0
7:P GridFrequency 3.1
8:P GridStripThickness 0.065

An anti-scatter grid can be simulated following the guidelines of Day&Dance(1983) [3].

  • UseAntiScatterGrid: If True, an anti-scatter grid defined as Day&Dance(1983) [3] will be considered.

  • GridRatio:

  • GridFrequency:

  • GridStripThickness: Thickness of the Strips in mm.

Note

The attenuation coefficients of the Stip Material and the Inter-Strip Material can be modified at `plug-ins/resources`.
